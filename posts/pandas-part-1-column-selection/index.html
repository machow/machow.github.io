mpg = r.mpg
from toolz import curry
type(mpg)
# moving column is tedious and requires
#  * multiple lines of code
#  * 
cols = list(mpg.columns)
cols.insert(0, cols.pop(cols.index("class")))
mpg.loc[:, cols]
cols = ["class", *[x for x in mpg.columns if x != "class"]]
mpg.loc[:,cols]
@curry
def move(indx, col_name, df):
  cols = list(df)
  cols.insert(indx, cols.pop(cols.index(col_name)))
  return df.loc[:, cols]
# renaming isn't too crazy (but notice the extra columns argument)
mpg.rename(columns = dict(car_model = "model"))
# combining renaming with moving
(mpg
   .rename(columns = dict(model = "car_model"))
   .pipe(move(0, "car_model"))
   )

Misc

Spec

  • actions
    • add a
    • remove a
    • add a as b
  • works sequentially
  • remove overrides add
  • if only removes, add non-removed by default

Caveats

  • current rules mean select(a, -b) doesn’t move a to front, and produce all non-b columns
  • but allows select(a, everything(), -b)
